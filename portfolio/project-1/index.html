USB Conference Camera

Date

Client

Camera Vendor

Categories

Pcba, software development

Project Requirements

It requires a USB Camera solution supporting UVC (USB Video Class) and UAC (USB Audio Class) with Camera Sensor (with HDR/WDR capability). It also connects to a speaker. AEC algorithm should be applied since Echo from Speaker to Microphone exists.

Project Details

It embeds four microphones and a loud speaker, which composes a audio conference system. It is driven by one SoC. This SoC also has embedded ISP to connect to a sensor, which supports HDR.

The UAC and UVC profiles supports Windows/Ubuntu/MacOS.

It also support AI features. With 4K or 2K resolution of sensor, it support EPTZ feature.
